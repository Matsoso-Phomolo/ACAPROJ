# Module Map

| Folder | Purpose |
|---|---|
| `backend/app/api` | HTTP API routes |
| `backend/app/core` | configuration and system settings |
| `backend/app/domain` | dataclasses and domain models |
| `backend/app/services` | authentication, file, and cryptographic services |
| `backend/app/storage` | consistent hashing and storage routing |
| `backend/app/ids` | IDS event engine and rate limiting |
| `frontend/static` | professional single-page dashboard |
| `docs` | architecture and demo documentation |
| `backend/tests` | unit tests for algorithms and core services |
